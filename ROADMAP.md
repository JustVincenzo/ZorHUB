# Roadmap
> Pre-alpha roadmap.

## pα0.1 — Foundation

Goal: create the first private pre-alpha version of ZorHUB as a GTK4/libadwaita Linux desktop application.

### Scope

- Main app shell.
- Basic navigation.
- Home module.
- Performance module.
- Storage module.
- Components module.
- Settings module.
- About module.
- Basic metrics.
- Local config.
- Local logs.

### Out of scope

- ZorHUB Agent.
- D-Bus communication.
- Polkit authorization.
- Privileged actions.
- App installation.
- System repair.
- Advanced cleanup.
- Public release.
- Flathub submission.

## pα0.2 — Consent & Components

Goal: add the component model, user consent, local action registry and local history.

### Scope

- First-run onboarding.
- Background consent setting.
- Components Manager.
- Action Registry for safe non-privileged actions.
- History module.
- Safer settings structure.
- Optional dependency detection.

### Out of scope

- Real ZorHUB Agent.
- Real D-Bus communication.
- Real privileged actions.
- Package installation or repair.
- System service management.
